               + admin campaign preset preview thumbnails

  [SUPPLIED] campaign-ube-taro.png     (preset: ube-taro)

Image resolution order at render time (CampaignAd / CampaignImage):
  1. Admin custom hero_image_url (only when explicitly provided)
  2. Bundled local image mapped by preset_key (the six above)
  3. Existing preset accent + emoji fallback
  If a custom URL fails to load it falls back to the bundled image, then accent.

DARK CAMPAIGN ASSETS:
