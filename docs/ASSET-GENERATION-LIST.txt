IMPORTANT NAMING RULES
- Product image filenames = the product's name LOWERCASED with spaces -> hyphens
  (this is how src/lib/productImages.ts maps them). Add the matching require()
  line there after dropping the file in.
- Campaign banner filenames are fixed and mapped in src/lib/campaignImages.ts.
- All PNG unless noted. Keep file sizes reasonable (<400 KB each ideally).
